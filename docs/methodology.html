1. Overview

The US Construction Materials Intelligence Report is a programmatically generated HTML dashboard covering the supply chain for cement and supplementary cementitious materials (SCMs) in the United States. It is designed to update automatically when underlying data is refreshed — every chart, statistic, and map marker is generated at build time from structured data sources rather than being manually maintained.

The report covers five material categories:

  • Portland cement — the dominant binding agent in US concrete
  • Ground granulated blast furnace slag (GGBFS) — a co-product of iron making that substitutes 30–70% of cement
  • Fly ash — a coal combustion by-product used as a pozzolanic cement replacement
  • Natural pozzolans — volcanic materials (pumice, perlite, diatomite, calcined clays) used as SCMs
  • Silica fume — an ultra-fine by-product of silicon/ferrosilicon production

The supply chain for these materials intersects iron & steel production (slag), electricity generation (fly ash), mining (natural pozzolans), and maritime/rail logistics (imports). This report captures all five layers in a single deliverable.

2.1 USGS Mineral Commodity Summaries (MCS)

The USGS publishes the Mineral Commodity Summaries annually, covering ~90 mineral commodities. Each commodity chapter reports: US mine production, imports for consumption, exports, apparent consumption (production + imports − exports), and average price. The MCS is the authoritative US government source for non-fuel mineral statistics.

  • URL: https://pubs.usgs.gov/periodicals/mcs2024/ (replace year)
  • Format: PDF (annual), one PDF per commodity chapter
  • Cadence: Published each January for the prior calendar year (with preliminary estimates)
  • Access: Free, public domain (US government work)

2.2 AIST Steel Mill Directory

The Association for Iron & Steel Technology (AIST) publishes a directory of steel production facilities in North America. It includes plant type (blast furnace, EAF minimill, rolling, finishing), annual capacity, and geographic coordinates. The blast furnace facilities in this directory are the primary source of granulated slag — a co-product of iron reduction that, when quenched and ground, becomes GGBFS.

  • Format: GeoJSON (converted from AIST directory, 68 facilities, 56 KB)
  • Included file: data/facilities_steel_mill.geojson
  • Filter applied: Only facilities with "blast_furnace" in the facility_types property are shown on the map — these are the 8 integrated BF-BOF mills that produce pig iron and, as a by-product, slag

3. Build Pipeline

The report is built by a single Python script: build_report.py. The build takes under 2 seconds and produces a fully self-contained HTML file.

[1/3] Load USGS data
      → Open DuckDB connection to data/usgs_minerals.duckdb (read-only)
      → Query mcs_salient_stats for each commodity/metric/year range
      → Aggregate to single median value per year (see §4)

[2/3] Load AIST mill locations
      → Parse data/facilities_steel_mill.geojson
      → Filter: country = "US", facility_types contains "blast_furnace"
      → Build JavaScript-embedded GeoJSON for Leaflet map

[3/3] Build HTML
      → Render all data into Chart.js dataset arrays (inline JavaScript)
      → Render facility coordinates into Leaflet marker code
      → Write single standalone .html file to output/

The generated HTML has no runtime data dependencies — all data is embedded as JavaScript literals. Only the map background tiles (CartoDB DarkMatter via CDN) and Chart.js / Leaflet / Google Fonts require an internet connection.

4. USGS Data Methodology

4.1 Database Construction

The usgs_minerals.duckdb database was built by a scraper-parser pipeline that:

  1. Downloaded all available MCS PDF files from USGS servers (37 commodities, 4,238 files, ~1.86 GB of PDFs)
  2. Parsed each PDF using text-block extraction to locate salient statistics tables
  3. Extracted year, metric name, value, and unit for each row
  4. Loaded results into the mcs_salient_stats table (32,344 records)

4.2 Multi-Record Deduplication

A key challenge: the USGS publishes multiple PDF editions covering overlapping years (preliminary estimates get revised in subsequent publications). This means any given commodity-year combination may appear 3–15 times in the database with slightly different values.

Resolution: For each commodity/metric/year, the build script filters to a plausible physical value range (e.g. cement production must be between 75,000–100,000 thousand metric tons), then takes the statistical median of all qualifying values. This produces a stable, revision-resistant estimate.

Example query for cement production:

SELECT year, value
FROM mcs_salient_stats
WHERE commodity = 'cement'
  AND metric = 'production_us'
  AND year BETWEEN 2020 AND 2024
  AND value BETWEEN 75000 AND 100000   -- plausibility filter (thousands MT)
ORDER BY year

→ Multiple values per year, e.g. 2022: [92000, 91500, 92300]
→ Median → 92000 → 92.0 MMT displayed in chart

4.3 Unit Normalization

USGS MCS values are typically reported in one of three scales:

  • Thousand metric tons — most production figures (cement, slag, pumice)
  • Metric tons — some price and smaller commodity figures
  • Dollars per metric ton — price series

The build script applies commodity-specific divisors to convert all production values to million metric tons (MMT) for display consistency. Prices are displayed in USD/MT.

4.4 Cement-Specific Notes

  • USGS reports Portland and masonry cement combined — the dominant form (~95%) is Portland cement
  • US apparent consumption is typically 15–22% higher than domestic production due to imports
  • Price data reflects the average value at the plant gate (not delivered price)

5. Facility & Location Data

5.1 Cement Plants (50 facilities — curated)

US cement plant locations and clinker capacities are sourced from PCA member data, USGS Cement commodity reports (which list major producers by state), and cross-referenced against public SEC filings from the major producers (Lehigh/Heidelberg, CEMEX, Martin Marietta, Summit Materials, Ash Grove/CRH). Coordinates are centroid estimates for the plant county.

Note: Cement plant data does not update automatically. The PCA does not publish a free machine-readable facility database. Capacities shown reflect industry conditions circa 2023–2024 and should be verified against the PCA Market Intelligence Report for precision work.

5.2 Import Ports (24 facilities — curated)

US cement-capable import terminals are identified from:

  • USGS annual cement commodity reports (list importing states/ports)
  • USACE waterborne commerce statistics (cement/clinker tonnage by port)
  • Panjiva/S&P Global trade data (vessel-level clinker import records)

Port volumes shown are indicative of relative import activity — not exact tonnage — as publicly available port-level cement tonnage is not systematically reported.

5.3 Blast Furnace Mills (live from AIST GeoJSON)

The 8 integrated blast furnace mills shown on the map are queried live from data/facilities_steel_mill.geojson. These are the only US facilities currently producing molten slag (a by-product of reducing iron ore to pig iron in a blast furnace). When quenched with water, this slag granulates and can be ground to GGBFS.

Active US BF-BOF integrated mills as of 2024:

  • US Steel — Gary Works (IN), Mon Valley Works (PA), Great Lakes Works (MI)
  • Cleveland-Cliffs — Burns Harbor (IN), Cleveland (OH), Middletown (OH), Dearborn (MI)
  • NLMK USA — Portage (IN)

5.4 Natural Pozzolan Deposits (8 deposits — curated)

Deposit locations are derived from USGS Mineral Resources Data System (MRDS) and USGS Professional Paper 1679 (Natural Pozzolans in the Western United States). All significant commercial-grade natural pozzolan deposits in the US are located in the western states (Oregon, Idaho, Nevada, California, Arizona, New Mexico) due to the concentration of Quaternary volcanic activity.

6. Commodity Coverage Detail

6.1 Portland Cement

Cement is manufactured by heating limestone (CaCO₃) and clay minerals to ~1450°C in a rotary kiln, producing clinker. Clinker is then ground with ~5% gypsum to produce Portland cement. The US has ~96 plants producing ~88–93 MMT/year, meeting ~80% of domestic demand; the remainder is imported primarily from Canada, Turkey, Vietnam, China, and South Korea.

6.2 Iron & Steel Slag / GGBFS

Blast furnace slag is produced at approximately 250–350 kg per tonne of pig iron. With ~25 Mt of pig iron produced annually at US integrated mills, approximately 6–9 Mt of slag is generated. Around 70% of this is granulated (water-quenched → GGBFS feedstock); the remainder is air-cooled and sold as aggregate or road base. GGBFS reduces cement content by 30–70% in concrete mixes, lowering embodied carbon by a similar proportion.

6.3 Fly Ash

Fly ash is captured from the flue gas of coal-fired power plants. US fly ash supply has declined ~40% since 2010 as coal plant retirements accelerate — from ~70 Mt available annually to ~45 Mt. Not all captured fly ash meets ASTM C618 specifications; usable supply is approximately 30–35 Mt/year, of which ~18–20 Mt is consumed as a cement replacement. Supply constraints are the primary growth challenge for fly ash-based SCMs.

6.4 Natural Pozzolans

Natural pozzolans include pumice, volcanic ash, calcined clay (metakaolin), calcined shale, diatomite, and perlite. US production is modest (~0.5–1.0 Mt/year total for SCM applications) but growing as fly ash supply tightens. Key producers: US Pumice (Pumice Products Company), Hess Pumice (ID), Dicalite Management Group (diatomite).

7. Limitations & Caveats

  • USGS MCS preliminary data: The most recent year in MCS (e.g., 2024) reflects preliminary estimates that will be revised in subsequent publications. Figures for the current year should be treated as ±5% estimates.
  • Fly ash — no live data: The USGS does not publish a dedicated fly ash MCS chapter. Fly ash statistics shown are based on ACAA (American Coal Ash Association) survey data and EPA RCRA reporting, cross-referenced against academic literature. These are curated estimates, not live database queries.
  • Cement plant capacities: Clinker capacities reflect publicly reported figures, not PCA survey data (which is proprietary). Actual utilization rates vary significantly by plant and year.
  • GGBFS market share: GGBFS market size (consumption as cement replacement) is modeled from slag production (USGS) minus air-cooled uses (road base, aggregate), with a slag-to-GGBFS yield factor of ~0.95. This is an estimate — no government agency tracks GGBFS consumption directly.
  • Map coordinates: Cement plant coordinates are county-centroid approximations in some cases; they should not be used for precise logistics routing without verification.

8. Setup & Reproduction

8.1 Requirements

  • Python 3.11 or higher
  • duckdb Python package (≥ 0.9.0)
  • Internet connection for map tiles and CDN assets when viewing the report

8.2 Installation

git clone https://github.com/theshipsagent/construction-materials-intel
cd construction-materials-intel
pip install -r requirements.txt

8.3 Building the Report

python3 build_report.py
# Output: output/construction_materials_intelligence_report.html

Open the output file in any modern browser (Chrome, Firefox, Safari, Edge).

8.4 Refreshing USGS Data

The data/usgs_minerals.duckdb database is a snapshot. To refresh it with the latest USGS publications, the full USGS Intelligence pipeline (not included in this repo) must be re-run. Contact the repository maintainer for refresh procedures or to request an updated database snapshot.
